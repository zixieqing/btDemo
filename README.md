# btDemo
## 介绍
cocos2dx-lua 做一个行为树demo
1.能够支持A* 寻路算法
2.tiledMap做地图相关逻辑
3.行为树包含一些基础功能（人物角色自动打怪，AI怪物攻击人物）
cocos 版本：cocos2d-x-3.17.2
